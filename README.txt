# CSC2002S - Assignment 2
# Concurrent Programming a word game
## Goal
### Use concurrency to safe parallelism in wordApp with no deadlocks

## Part 1: Program
### Build
```unix
$ make
```
### clean
```
$ make clean
```
### run Program
```
$ make run total_words=<total_words> words_on_screen=<words_on_screen> dict_file=<dict_file>
```

## Documentation
### generate Javadocs
```
$ make docs
```
### clean Javadocs
```
$ make clean-doc
```
